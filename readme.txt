Nombre del proyecto: Visualizacion de estadisticas de productividad 1.0

Funcionamiento:
Graficar los datos relacionados con las estadisticas de productividad desde 1978 hasta 2022.
Pasos:
1. Leer el conjunto de datos original
2. Limpiar el conjunto de datos (valores nulos, valores atípicos, etc.)
3. Guardar y cargar el nuevo conjunto de datos limpio
4. Graficar

Ténicas utilizadas:
-Limpieza de datos
-Análisis de datos
-Visualización de datos

Conjunto de datos utilizado: productivity-statistics-1978-2022
Enlace: https://www.stats.govt.nz/assets/Uploads/Productivity-statistics/Productivity-statistics-19782022/Download-data/productivity-statistics-1978-2022.csv

Tecnologías utilizadas:
-Lenguaje de programación Python==3.12.0
-Entorno de desarrollo integrado PyCharm
-Editor de texto Visual Studio Code
*Las librerías de Python utilizadas en el proyecto se encuentran en el archivo requirements
